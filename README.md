## How to install

### Option 1 (Download):

Download the `rmaki-obsidian.css` file and copy into your themes folder

### Option 2 (Build):

1. Clone the repo
2. Navigate to folder

```bash
cd ./rmaki-obsidian/
```

3. Install deps

```bash
yarn install
```

or

```bash
npm install
```

4. Run build

```bash
yarn build
```

or

```bash
npm run build
```
